Marker Setup

Tap anywhere on the camera view to mark each point in sequence:

  1. Starting ball position
  2. First marker - Left point (3ft from ball, ½ft left of path)
  3. First marker - Right point (1ft from left point, across path)
  4. Second marker - Left point (6ft from ball, ½ft left of path)
  5. Second marker - Right point (1ft from left point, across path)

Tips:

  • Place physical markers (tees, coins, etc.) at these points
  • Ensure camera can see all markers clearly
  • Keep phone steady during setup
